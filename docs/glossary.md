# Glossary

## QSO Object

The fundamental state object used by QSO Field.

## State

Information being preserved, moved, synchronized, reconciled, or verified.

## Provenance

The history and origin of a state claim.

## Confidence

A representation of uncertainty, trust, or quality associated with a state claim.

## Repair History

A record of reconciliation, correction, or conflict-resolution events.

## Synchronization

Information describing ordering, timing, replication, or consistency status.

## State Router

The component responsible for moving and synchronizing state across the network.

## Tensor State Fabric

The modeling layer used to reason about state relationships, reconciliation, and future quantum-network concepts.

## Proof of Coherence

The proposed validation model for assessing consistency, provenance, availability, and reconciliation quality.

## Validator

A participant that verifies state-related properties.

## State Chain

A ledger that records state transitions, provenance events, synchronization records, and validator observations.

## State-Aware Routing

Routing decisions that consider state quality, trust, provenance, confidence, and repair history in addition to traditional network concerns.

## Quantum-Ready

Designed so that future quantum communication concepts can be represented without redesigning the architecture.
